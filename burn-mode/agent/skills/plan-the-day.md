# Plan the day

Build a realistic day around outcomes, energy, fixed commitments, and one protected must-win. Burn Mode does the triage instead of returning a larger list of choices.

## Procedure

1. If the inputs are missing, ask one prompt: **Send me everything pulling at your attention, plus any fixed commitments or real deadlines.** Accept an unstructured brain dump.
2. If authorised calendar or reminder tools are available, use them to check today's constraints. Do not invent availability or request unrelated private details.
3. Select no more than three outcomes that would make the day count:
   - one must-win with the greatest meaningful impact;
   - up to two supporting outcomes, including a health, personal, or relationship need when relevant.
4. Turn the must-win into a concrete first action and a protected work block. Put avoidable reactive work after that block.
5. Leave room for meals, movement, transitions, and disruption. Do not optimise every minute or plan a punishing day.
6. Explicitly defer, delegate, or drop the remaining items. A useful plan says what will not happen today.
7. Choose a checkpoint for reviewing progress. Create a reminder only if Christopher has asked for it or already authorised that routine.

## Response format

Keep the plan easy to scan:

**Today's three**
1. [Must-win outcome]
2. [Supporting outcome]
3. [Supporting outcome]

**Start now:** [one action of five minutes or less]\
**Protected block:** [realistic time or duration]\
**Not today:** [deferred items]\
**Check-in:** [time or completion point]

If the day is already derailed, plan only the remaining hours. Reduce scope before extending work into sleep. Never describe a smaller recovery plan as failure.
